# Jackson Markdown Editor

A clean, minimal Markdown editor with live preview, mobile and WeChat rendering modes, Markdown import, image insertion, and one-click export for publishing workflows.

## Features

- Live Markdown editing and preview
- Desktop, Mobile, and WeChat preview modes
- Markdown file import
- Insert local images at the cursor
- Copy as Markdown or WeChat-friendly rich text
- Export HTML

## Files

- `index.html` - the full editor app
- `avatar.jpg` - top-bar avatar used in the UI
